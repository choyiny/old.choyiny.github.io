Factorizing Semiprimes

This is a simple java program that factors semiprimes using multiple methods: Brute force, Fermat's Factorization method, and Pollard's Rho method. It aims to "crack" RSA keys. This is part of my IB Mathematics Higher Level Internal Assessment.
