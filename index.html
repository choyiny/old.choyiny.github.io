Contributing to
The Open Source World

My first open source project was a Minecraft Bukkit Plugin ChatMaster, made for making lives of server administrators easier in managing chat. I made this to learn the java programming language and developing Bukkit plugins.

I usually put the code I use to learn the language to Open Source. For my IB DP Mathematics HL Internal Assessment, I chose to write a program to Factorize Semiprimes, which is a proof of concept to "crack RSA". My blog, created with Hugo, is also put to open source, as well as the coding excercises I attempted for Google Code-in 2015. More Open Source projects can be viewed on my github page.

I have contributed to various Open Source projects such as @FOSSASIA's privly-chrome, gci15.fossasia.org, @Google's Material Design Lite, Glowstone, @FGMEMBERS' A320neo and more...
